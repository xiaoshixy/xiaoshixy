# 合并单元格

点击查看 
<table border="1px" width="400px" height="200px">
    <tr>
        <td>1</td>
        <td>2</td>
        <!-- 行合并口诀： 后面的单元格注释掉 
        在前面的单元格使用rowspan属性进行合并
        合并几个写几个 -->
        <td rowspan="2">3</td>
    </tr>
    <tr>
        <td>1</td>
        <td>2</td>
        <!-- <td>3</td> -->
    </tr>
    <tr>
        <!-- 列合并口诀：两个单元格随意注释其中一个，
        在另一个单元格使用colspan属性进行合并
        合并几个写几个 -->
        <td colspan="2">1</td>
        <!-- <td>2</td> -->
        <td>3</td>
    </tr>
</table>

# 超链接

href这个属性的值是什么

href属性的值：请求路径

浏览器向服务器发送，叫做：请求（request）

B--向发送数据叫做请求--S

服务器向浏览器发送，叫做：响应（reponse）

S--向发送数据叫做响应--B

<!-- href属性指定链接路径 -->
<!-- target属性_blank新窗口打开_self当前窗口打开 -->
<a href="https://www.jd.com" target="_blank">123</a>
1
2
3

target属性：

  1. 可不写
  2. _self 当前窗口
  3. _blank 新窗口
  4. _parent 父窗口
  5. _top 顶级窗口
<!-- inner frame 嵌入子窗口 -->
<iframe src="12.列表.html" frameborder="0"></iframe>
<iframe src="https://xiaoshixy.gitee.io/" frameborder="0"></iframe>

# 表单

表单收集用户数据，提交表单，可以向服务器发送请求，并且可以携带用户填写的数据

  • 向服务器发送请求：
    • 直接地址栏输入url
    • 超链接
    • 填写表单
点击查看 
<!-- 
form标签属性： action：指定请求路径，提交给谁
method：get:提交数据显示地址栏(默认)；post:提交数据不会显示地址栏
当使用超链接、url都是get请求
-->
<form action="http://www.baidu.com" method="get">
    <!-- input标签是输入域 -->
    <!-- submit 提交按钮 置于form表单内部，提交当前表单-->
    <!-- value:设置文本  -->
    <input type="submit" value="提交" /><br />
    <!-- button 普通按钮，无提交能力 -->
    <input type="button" value="百度按钮" /><br />
    <!-- text 文本框 -->
    <input type="text" value="xxx" /><br />
    <!-- checkbox 复选框 -->
    <input type="checkbox" name="" id="" /><br />
    <!-- radio 单选框 -->
    <input type="radio" /><br />
    <!-- password 密码框 -->
    <input type="password" />
