终于通过前三关了，科目四冲啊！离小本本还差最后一步！

# 文明驾驶技巧

# 判断题

  1. 可以不，错误
  2. 慢慢减速、缓慢减速、逐渐不妨碍，一般都对
  3. 交通陋习：随意开车，加塞，抽烟，非机动车停车，都错
  4. 加速、快速、迅速一般是错的，迅速让行除外
  5. 避免、以免，一般都对
  6. 看到电视，信号干扰；遇到动物，鸣喇叭，都错
  7. 白天实景图，判断一般都对，多选全选
  8. 合理使用、轻按喇叭正确，连续错误
  9. 立刻，立即，马上，直接，只要，要急，都错
  10. 易，一般对，大车除外
  11. 提前，主动都是对的
  12. 空挡，加速车道停车，等待救援，都错
  13. 借道都错，虚线除外

# 选择题

  1. 审题：注意是单选题还是多选题
  2. 多选题：白天实景图的 ABCD一般都是对的
  1. 汇入主干道，找后入，保证安全，不妨碍
  2. 注意观察、减速慢行、减速靠右【遇到警车，靠左侧减速让行】、避让、让行、礼让，一般直接选
  3. 平稳停车、大间距、大车距、保持距离，直接选
  4. 四个选项一个减速直接选，有停车优先选
  5. 堵车找依次、排队
  6. 车前方有人，找停车 ⭐️

# 安全常识及规定

# 行车前检查

  • 逆时针绕车检查
  • 确认出车方向的安全
  • 下车观察侧后情况

# 安全带作用

  • 减轻伤害【只有一个作用】

# 安全头枕

  • 头枕找头部

# 行车规定

  1. 路口内堵车，停止线外，停车等待

  2. 有障碍的一方给无障碍的一方让行（有障碍一方已行驶至中途，无障碍车辆应当礼让）

  3. 口五站三

  4. 违法行为

  5. 三原则，【一二三】对

  6. 规定限定速度，【不一致】错

  7. 车前方校车问题【一二停车，三减速】

  8. 顺序通行三原则：

    • 转弯让直行
    • 右转弯让左转弯
    • 同样直行让右侧先行

  9. 关于超车

    • 只能左侧超车，尽快超越，减少并排时间
    • 选择看到超车，找放弃【特殊车辆、特殊道路】

# 灯光使用

# 基本使用

  1. 危险报警灯
    • 停车或发生故障，开危险报警灯
    • 雾天开雾灯、危险灯
  2. 近光灯
    • 有交替直接选，没有选近光灯（动物除外）
    • 会车150米以外改近光
  3. 远近光灯：判断题
    • 近光都对，远光灯都错，远近都对，分开错（货车除外）
    • 超车交替远近
  4. 灯光作用
    • 有利自己，方便其他
    • 会车切换远近观察前方
  5. 环岛⭐️
    • 进入不开、驶出右转灯
  6. 示廓灯
    • 黄昏开

# ⭐️根据灯光判断道路与地形

  1. 由远变近

    • 靠近上坡，急弯，低谷地段

  2. 由近变远

    • 由弯道进入直线行驶或者下坡道已由陡坡进入缓弯

  3. 离开地面

    • 急弯或大坑，坡顶

# 特殊天气

  1. 灯光使用

    • 黄昏开示廓灯
    • 雾雨雪天气，开近光灯

  2. 雨刮器

    • 刮水器，有雨、水使用

  3. 大风天气

    • 看到大风，双手紧握方向盘，控制方向

  4. 低能见度行车

    • 开启前大灯、示廓灯和后位灯，雾天开雾灯

  5. 关于大雾特殊天气能见度车距车速

    • 261（能见度、车速、最小车距）
      • 开启雾灯、近光灯、示廓灯和前后位置灯
    • 145
      • 开启雾灯、近光灯、示廓灯和前后位置灯、危险报警灯
    • 52离
      • 开启雾灯、近光灯、示廓灯和前后位置灯、危险报警灯

# 特殊路段

# 特殊道路规定

  • 特殊道路最高限速30
  • 特殊路段不正常错误

# 积水路段

减速慢行

# 漫水桥

  • 停车观察，确认安全匀速通过，不换挡停车
  • 车辆涉水，制动效果降低
  • 间断轻踏刹车

# ⭐️坡道制动

  • 上坡【推迟】，下坡【提前】
  • 下坡找【两重】，上坡找【三无】（多选题）

# ⭐️坡道停车

上坡后溜：上，左（方向盘）

下坡前溜：下，右（方向盘）

# 会车问题

  • 山路：靠山让临崖
  • 坡道：下坡让上坡

# ⭐️易滑路段

  • 刚下雨最滑
  • 泥泞路段/水滑现象，用加速踏板控制车速
  • 避免刹车、低速行驶、稳握方向
  • 驱动轮垫石头

# 冰雪路面

  • 制动延长
  • 前车爬坡，等通过
  • 找车辙
  • 无车辙找参照物（路杆，树木）

# 坡道行车

  • 禁止空挡
  • 长时间连续制动，制动效果下降
  • 挂低速档利用发动机控制车速

# ⭐️立交桥转弯

  • 都是向右转
  • 右转：桥前右转
  • 左转：桥后右转

# 速度问题

# 限速问题

  • 特殊路段，速度不超30
  • 城3公4
  • 城5公7【有中心线】

# 换挡技巧

  • 车速降低前换挡
  • 上坡换低速档

# 高速技巧

  • 车速范围：60~120
  • 车速高于100，找100米；低于100，不低50
  • 高速状态紧急避险，先减速再动方向
  • 高速公路，非紧急不占用应急车道
  • 发生事故要到路肩栏杆以外

# 高速不同车道限速范围

  • 两条：找100
  • 三条：9、11
  • 四条：6、8、10

# 现场急救

# 救助原则

  • 事故找协助
  • 先救命后治伤
  • 先检查呼吸

# 骨折伤员

  • 不随便移动骨折部位，应固定，不要软担架运送
  • 受伤严重能不移动就不移动

# 失血伤员

  • 先止血
  • 近心端动脉，不能用细绳子包扎
  • 失血过多保暖

# 中毒伤员

  • 先移送空气新鲜地方
  • 不要仰卧位

# 烧伤伤员

  • 冷水喷洒
  • 水泡用塑料袋覆盖
  • 淡盐水

# 交警手势

# 判断题：

人物的头看向我们都是对的，侧身都是对的，手有重影都是对的

# 选择题

  • 选项中看到四个字的选项直接选
  • 没有四个字的选项，找带有“转弯”两个字的选项

# 新规题目

  1. 不按规定使用灯光，设置警告标志，一次记3分
  2. 遇人行横道，未减速、避让、礼让行人，罚款200元，一次记3分

# 货车专项

  1. 绿灯，进入左转待转区
  2. 变更车道，提前3s开启转向灯
  3. 右转弯遇右侧有车，在右侧车后向右变道
  4. 横向间距：车速每小时100公里，最小距离1.5米
  5. 警告标志：高速不少于150，普通不少于50
  6. 白色标线：车行道分界线
  7. 反光标识：后部、侧身
  8. 止血法：紧急情况止血，用压迫法；关节出血，屈肢加垫止血法
  9. 止血带止血上臂位置：出血点上三分之一
  10. 夜间通过照明不好的路段
    • 车速高于30公里，可以开远光
    • 车速低于30公里，可以开近光
  11. 湿滑路面紧急制动，会出现横滑或侧滑事故
  12. 后车跟车太近可以轻踩刹车提示后车，保持安全间距
  13. 胎压是指轮胎正常情况的最大充气压力值
  14. 停车常识：先拉手刹，再摘挡，再送脚刹
