# 信息

# 概念

💠是一种客观事物，能够用来消除不确定性

# 信息的特征

点击查看

🔸客观性——主观信息必然转换为客观信息

🔸依附性——不能脱离物质独立存在

🔸传递性——时间上传递是存储；空间传递是转移或扩散

🔸转换性——将信息转化为物质或能量

# ⭐信息的7个质量属性

口诀

安全靠精整及经验

点击查看

🔹安全性——可被非授权访问

🔹可靠性——可信任

🔹精确性——精准程度

🔹完整性——全面程度

🔹及时性——发生间隔长短

🔹经济性——成本在可接受范围

🔹可验证性——证伪或证实
