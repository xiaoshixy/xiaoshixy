# 测试方法

点击查看

🅰️静态测试

  • 不运行程序
  • 用检查单、桌前检查、代码走查、代码审查等进行测试

🅱️动态测试

  • 运行程序

  • 白盒测试

    • 完全清楚内部结构和算法，用于“单元测试”
    • 常用技术
      • 逻辑覆盖

    主要方法: 控制流测试、数据流测试、程序变异测试

  • 黑盒测试

    • 完全不考虑结构和算法，用于“集成、系统、确认测试”
    • 常用技术
      • 等价类划分
      • 边界值分析

# 分类

点击查看
  • IaaS（基础设施服务）
    • 包括服务器、网络、硬盘存储和数据中心等基础设施
    • AWS、Microsoft Azure、阿里云、ECS服务器、云存储

    • 操作系统、数据库、 WEB应用等变服务
  • SaaS（云的应用）
    • 应用软件、组件、工作流等变服务
    • 百度云盘（个人级 SaaS）、钉钉（企业级 SaaS ）、53KF 客服系统（企业级 SaaS ）

# 企业系统规划BSP方法

点击查看

1、项目确定；

10、成果报告

# 信息系统规划工具

  1. 过程/组织（P/O）矩阵
    • 组织对于某个过程是负责决策、还是主要涉及、还是不涉及
  2. 资源/数据（R/D）矩阵
    • 某一类数据包含哪些资源的具体信息
  3. 过程/数据类CU矩阵
    • 过程要使用某一方面的数据，还是会创建该数据
