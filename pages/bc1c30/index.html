炎热夏天苦逼练车的小可怜，记录练车心得

# 调整座椅位置

  1. 先调靠背倾角
  2. 再调座椅高低
  3. 最后座椅前后
点击查看

  • step1: 打开车门，将座椅拖到最后面，便于上车；

  • step2:上车后，左手顺着靠背左侧往下摸到扳杠，向后按住扳杠，身体前倾，靠背自动贴到靠背上，松开扳杠，身体自然后仰， 靠背自动锁死时位置；

  • step3:座椅左侧扳杠调整座位高低，往上一下一下按座位升高。座椅高低标准：头顶距离车顶一拳距离；

  • step4:调整座椅前后，一只手提起横杠不要松，另一只手握住方向盘，座椅可以前后移动。

    调整合适位置：刹车踩到底舒适受力为准，大腿与小腿夹角45度；

# 左右后视镜调整

先调整座椅，再去调节左右后视镜

  • 左后视镜旋钮指向L

    • 先调左右
      • 调整车身占后视镜约1/3位置
    • 再调上下
      • 后门把手调至后视镜上沿

  • 左后视镜调整好示例图：

  • 右后视镜旋钮指向R
    • 先调左右
      • 调整车身占后视镜约1/3位置
    • 再调上下
      • 后门把手调至后视镜上沿
  • 调整完毕后 旋钮指向O

# 方向盘最高点或者车标对正参照物

  • 标记与参照物横向距离：D

  • 当标记与参照物对齐，先不回方向

  • 直到标记在右侧与参照物横向距离：D

  • 调整方向盘标记再次与参照物重合，此时回正方向

# 右打满太早，车头向左偏斜

调整方法：

确定右后轮能安全进库：车身没有盖住库的右前角

  1. 回正方向盘，继续看右后视镜；
  2. 库的右前角从后视镜消失的时候，再次向右打死；
  3. 当从左后视镜看到库的左后角出现，回正方向。

# 右打满稍早

调整方法：

  • 回半圈，当从左后视镜看到库的左后角出现，回正方向

# 一把入库

不需要调整，当从左后视镜看到库的左后角出现，回正方向

# 车速快，打方向晚，打方向慢，车头偏右

右宽左窄

调整方法：

看左后视镜能看到库的左前角还能补救

  1. 保持右打满继续往后倒，观察左后视镜看到左后轮可以安全进库；
  2. 继续看右后视镜，车身盖住右后角，回正方向盘；

  1. 后续修正方向，车尾哪边宽，就往哪回方向，90度，调整好 回正

# 侧方停车

# 操作步骤

  • 方向盘对齐中心线前进，肩膀对齐点位，停车
  • 挂R档，迅速右打满，观察左后视镜，第二个墩子一出现，迅速回正
  • 后轮压边线，迅速左打满，千万不要中途停车
  • 头探出窗户，观察前轮，前轮进库位，停车
  • 换D档，开启左转向灯，观察左后视镜，第二个墩子出现，迅速回正
  • 观察车头不要贴近黄线，往右回一圈，摆正车身后，出库

# 直角转弯

# 操作步骤

  • 领线对齐点位，摆正车身
  • 提前打开左转向灯
  • 进入直角转弯区域，关闭转向灯
  • 肩膀对齐打满点位，迅速左打死
  • 直至摆正车身，迅速回正

# 窄路掉头

  • 肩膀对齐中间线，领线对齐点位前进，迅速左打死
  • 摆正车身后，车前盖即将盖住黄线，迅速左打死
  • 看左后视镜，后轮过中间线，停车
  • 换R档，往右打死，观察左后视镜位置过中心线，停车，换D档，往左打死，直至摆正车身

# S弯

  • 当后视镜外端与左线重合时，往左打一圈方向

  • 微调方向，始终让外端顶到左线

  • 提前判断两圆交点

  • 当左线走到镜子根部，回正方向
