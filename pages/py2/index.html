# 数据类型

可变类型：列表、字典、集合

不可变类型：整型、浮点型、字符串、元组

所谓可变与不可变：数据能直接进行修改就是可变，否则就是不可变。

科学解释：

可变类型：当变量值改变，id内存地址不变。

不可变类型：当变量值改变，id内存地址就改变了。

使用id(变量名)可以查询id内存地址

# 切片

语法：序列[开始位置下标：结束位置下标：步长]

结束位置下标：不包结束位置下标对应的数据，正负整数均可

步长：选取间隔，正负整数均可，默认为1

注意：省略起始值，默认从0开始；省略结束值，默认到最后。

print(变量名[ : -1] )      #负1表示倒数第一个数据

# 字符串的常用操作方法

# find()

查询某个子串是否包含在这个字符串中,如果在返回这个子串开始的位置下标，否则则返回-1

语法：字符串序列.find(子串，开始位置下标，结束位置下标）

# index()

查询某个子串是否包含在这个字符串中，如果在返回这个子串开始的位置下标，否则报错。

语法：字符串序列.index(子串，开始位置下标，结束位置下标）

# count()

返回某个子串在字符串中出现的次数

语法：字符串序列.count(子串，开始位置下标，结束位置下标）

# replace()

注意：替换数次如果超过了子串的出现次数，就替换所有子串。省略替换次数就是全部替换。

语法：字符串序列.replace(旧子串，新子串，替换次数）

# split()

语法：字符串序列.split(分割字符，分割次数）

# join()

语法：字符或子串.join(字符串1，字符串2，字符串3，……）

# startswith和endswith

语法：

变量名.startswith（子串，开始位置下标，结束位置下标）

变量名.endswith（子串，开始位置下标，结束位置下标）

# 字典（可变型）

当下标（索引）不好用的时候！就要用到字典了，字典和数据顺序无关，所以字典不支持下标，字典是按key和value的方式成对出现，俗称键值对。

字典用大括号包裹，数据是键:值成对出现

# 空字典的创建

名字 = { }
名字 =dict( )
1

# 字典之查询

提示

查数据：注意只能用键查值，不能用值查键，因为键是唯一的，值可能是重复的。

# get()

语法： 字典序列名.get(键，随便写)

如果键存在，返回值。如果键不存在，返回默认值，默认值是你随便写的内容，如果省略了这个参数，返回None。

# values()

语法：字典序列名.values()

返回字典中所有的值

# items()

语法：字典序列名.items( )

可迭代对象（里面的数据是元组）,迭代就是重复反馈过程

# 集合

# 创建集合

提示

可以用{ } 或 set( )创建集合，但是创建空集合必需用set( )，因为{ }创建的是空字典

集合的特点：(1)自动去除重复数据 (2)顺序是随机的，所以不支持下标

# 集合之增加数据

语法：集合名.add(数据)

因为集合自动去重复，所以增加重复内容时不进行任何操作

# 集合之追加数据序列

语法：集合名.update(数据序列)

数据序列：列表，字符串，元组

# 集合之删除数据

集合名.remove(数据) 如果数据不存在，报错

集合名.discard(数据) 如果数据不存在，不报错
